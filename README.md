# Kiem Tra CK CT

## Project Overview
This project is a website for the restaurant "Nhà hàng Kiệt Anh Nhi". It provides information about the restaurant, its menu, and contact details.

## File Structure
- `index.html`: The homepage of the restaurant website, featuring an introduction and links to other pages.
- `menu.html`: A detailed presentation of the restaurant's menu, including various dishes and drinks.
- `about.html`: Information about the restaurant's history, mission, team members, and dining experience.
- `contact.html`: Contains the restaurant's contact information, including address, phone number, email, and a map.
- `style.css`: Styles for the website, defining layout, colors, fonts, and overall appearance.
- `README.md`: Documentation for the project, including setup and usage instructions.

## Setup Instructions
1. Clone the repository:
   ```
   git clone https://github.com/quanganhnek2k6/Kiem_tra_CK_CT.git
   ```
2. Navigate to the project directory:
   ```
   cd Kiem_tra_CK_CT
   ```
3. Open `index.html` in your web browser to view the website.

## Usage
- Navigate through the website using the links provided in the navigation bar.
- Explore the menu, learn about the restaurant, and find contact information.
